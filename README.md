# DnD Spellbook
A spellbook implementation for DnD players.

## Getting Started

```npm install```


### Features

* Checkout the spells from DnD 5th edition.

### On Deck Features to Implement

* Search for a specific spell
* Build a custom spellbook to use in your games.
